## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/carousel-reanimated.git
```

Go to the project directory

```bash
  cd carousel-reanimated
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run android
```
